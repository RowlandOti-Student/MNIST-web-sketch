MNIST digit sketch

This was my "hello world” project for building my very first neural network.
I was curious how a simple shallow network would perform on unseen data points that might be completely different from the training and validation data sets.
This led me to explore the idea of letting users interact with the neural network and see how the accuracy would change upon users intentionally trying to break the network by drawing weird looking digits.

Results are somewhat expected, although it’s surprising how well this simple multilayer perceptron manages to deal with the classification. It's hard to say how accurate the real-time guessing is without sufficient sample data. This is why currently I do not save the classification results to tell you how well it's doing overall.

If more sample data were available, it would be interesting to store all drawn digits and their correct labels to retrain the network over time. Even though this seems somewhat unnecessary for such a simple web app, this suggests that human interaction can provide useful information for machines to adapt to new scenarios.

For technical details, please see my Github repo above.
